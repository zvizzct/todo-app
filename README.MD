#  Todo-app Vite

This is a simple to do app built with Vite and vanilla javascript.

##  Getting Started

###  Install dependencies

```bash
npm install
```

###  Run the app

```bash
npm run dev
```

### Open the project in your browser

```bash
http://localhost:5173
```

## Production

###  Build the app

```bash
npm run build
```

### Deploy ```dist``` folder to your server
